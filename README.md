# UNP_PROJECT_BI_WITH_MS_EXCEL_EMPLOYEE_DATA

I completed a Bank Data Analysis project in collaboration with UNP using Power BI.Performed data cleaning and transformation through Power Query.Built data models and created DAX measures to analyze key KPIs such as deposits, loans, and balances.Developed interactive dashboards to track branch performance and customer segmentation.Generated meaningful insights to support data-driven financial decision-making.Implemented slicers and filters to enable dynamic and real-time data exploration.Analyzed transaction trends to identify patterns and performance gaps.Enhanced reporting efficiency by automating data visualization and KPI tracking.Conducted branch-wise and account-type performance comparisons.Identified high-value customers through segmentation analysis.Monitored loan distribution and repayment patterns.Created KPI cards to display real-time financial metrics.Optimized dashboard design for better user experience and clarity.Ensured data accuracy through validation and consistency checks.Presented findings with actionable recommendations for improved banking operations.
